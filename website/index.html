Namaste Duniya! 🙏

The classic "Hello World" in JugaadLang. Notice how natural it feels — bolo means "say" and agar/warna mean "if/else".

This is the first program every JugaadLang developer writes. By tradition, you must also say "Namaste" to your chai before running it.

bolo agar warna
.jug

Functions — banao aur chalao 🔧

banao means "create" — you create functions. wapas means "return/back" — the value comes back to you.

ghumo means "roam/iterate" — perfect for loops because you're literally roaming through the elements!

banao wapas ghumo agar
.jug

Classes — Ustad ka darbar 🎓

ustad means "master/teacher" — classes are masters who teach objects what to do.

khud means "self" — because in JugaadLang, you always know yourself. shuru means "start/begin" for `__init__`.

ustad banao khud bolo
.jug

Error Handling — Gadbad ho gayi! ⚠️

koshish means "try/attempt". gadbad means "problem/trouble" — because when there's an exception, there's always gadbad!

aakhir_me means "in the end" — the finally block that runs no matter what, like a responsible Indian parent.

koshish gadbad aakhir_me
.jug

Pattern Matching — agar_match & kaand 🎯

Structural pattern matching like Python 3.10. Use agar_match to match a variable or expression, and kaand for individual case matches.

Perfect for structured data, sequence unpacking, and complex conditionals with default catch-all matches.

agar_match kaand sahi wapas
.jug

JugaadWeb — API & Server 🌐

Build web servers and routing APIs easily using the native web library wrapper. Decorate your handler functions with route decorators like @web.agar_route.

Run your server by calling web.chalao(). It's that simple!

lao banao wapas chalao
